# fucking NJFU WIFI

## 简介

这是一个南京林业大学校园网自动化登入的脚本，使用jquery编写，建议配合油猴插件来使用

## 使用方式

- 在js文件中添加自己的账号和密码（建议）
  1. 首先安装油猴插件
  2. 将fucking NJFU WIFI 这个文件添加到脚本库中
  3. 根据注释，改变username和password的值
- 使用cookie
  1. 首先安装油猴插件
  2. 将fucking NJFU WIFI 这个文件添加到脚本库中
  3. 在第一次登入的时候会提示你没有cookie，需要手动登入之后才能自动添加cookie
